Project Overview
A personal link in bio page built with Python and Flask where users can add URLs and the app displays them as styled link cards.

Tech Stack
Python
Flask
HTML

File Structure
app.py for the main app
a templates folder for HTML files
a static folder for CSS.

Coding Conventions
using clear variable names
adding comments for complex logic
keeping functions small and focused.

Testing
after changes run python app.py to confirm the app starts without errors.
